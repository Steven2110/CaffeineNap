# Features:

- Up to date self curated beverages data
- Insightful widgets

# Requirements:
- iOS 16.0 or newer
- watchOS 9.0 or newer

# Used library:
- SwiftUI
- WatchKit
- CloudKit

- SwiftCharts
- WidgetKit
